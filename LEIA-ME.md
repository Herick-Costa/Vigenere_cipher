# Cifra de Vigenère em Python
## Este repositório contém um script simples em Python para criptografar mensagens usando a cifra de Vigenère.

## Como Usar
1. Clone este repositório ou baixe o arquivo vigenere_cipher.py.
2. Execute o script Python em seu ambiente local.
3. Insira a mensagem que deseja criptografar quando solicitado.
4. Insira a chave de criptografia quando solicitado.
5. O script criptografará a mensagem usando a cifra de Vigenère e exibirá a mensagem criptografada.

## O que é a Cifra de Vigenère?
A cifra de Vigenère é um método de criptografia que utiliza uma série de diferentes cifras de César com base em letras de uma palavra-chave. Foi inventada pelo criptógrafo francês Blaise de Vigenère no século XVI. A cifra de Vigenère é mais segura do que cifras de substituição simples, como a cifra de César, porque altera as letras da mensagem de acordo com uma palavra-chave.
